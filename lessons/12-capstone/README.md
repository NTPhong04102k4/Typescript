# 12 · Capstone

This topic pulls together everything from lessons 00-11 into a final push:
recursion and backtracking, dynamic programming, and three cross-topic
practice sets ordered by difficulty (Easy, Medium, Hard) that mix array,
string, linked list, stack, queue, heap, map/set, tree, and graph problems in
the same file. It closes with a decision-making cheat sheet for picking the
right data structure under pressure and a single complexity comparison table
for quick review. Work through 01-02 first since backtracking and DP show up
inside the mixed practice sets; 03-05 can be done in any order once those
land; 06 is a reference to revisit anytime.

Lessons:
- 01 — Recursion & backtracking primer
- 02 — Dynamic programming primer (arrays/maps as memo)
- 03 — Mixed practice set 1 (Easy, cross-topic)
- 04 — Mixed practice set 2 (Medium, cross-topic)
- 05 — Mixed practice set 3 (Hard, cross-topic)
- 06 — Final review: choosing the right data structure + engine perf cheat sheet
