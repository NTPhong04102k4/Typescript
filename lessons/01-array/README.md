# 01 · Array

Arrays are the foundation of most DSA patterns: a contiguous block of memory
indexed in O(1), which is why they dominate interview problems. This topic
walks from the raw memory model up through the classic array techniques —
two pointers, sliding window, prefix sums, binary search, matrix traversal,
and Kadane's algorithm — before finishing with in-place tricks and a
practice set. Each lesson builds on the previous one, so work through them
in order.
