# 03 · Linked List / Node

Linked lists trade the contiguous, index-addressable memory of arrays for a
chain of nodes connected by pointers, giving O(1) insert/delete at known
positions at the cost of O(n) random access. This topic starts with the
struct-like `ListNode` building block, builds singly and doubly linked
lists, then layers on the classic pointer-manipulation patterns — fast/slow
pointers, reversal, merging/sorting, and an LRU cache — before finishing
with a practice set. Each lesson builds on the previous one, so work
through them in order.

Lessons:
- 01 — Node fundamentals: struct-like type in TS
- 02 — Singly linked list implementation
- 03 — Doubly linked list implementation
- 04 — Fast & slow pointers (cycle detection)
- 05 — Reversal techniques (iterative & recursive)
- 06 — Merge & sort linked lists
- 07 — LRU cache via node + map
- 08 — LeetCode practice set: Linked List
