# 02 · String

Strings in JavaScript/TypeScript are immutable, UTF-16-encoded sequences
that look like arrays but behave differently under the hood. This topic
builds from the memory model up: traversal with two pointers, sliding
windows, hashing/anagram tricks, efficient string construction, pattern
matching with KMP, tries for prefix search, and stack-based parsing. Each
lesson pairs a theory file with a runnable TypeScript companion containing
exercises and real LeetCode problems, closing with a dedicated practice set.

Lessons:
- 01 — String fundamentals & immutability in JS/TS
- 02 — Traversal & two pointers (palindromes)
- 03 — Sliding window on strings
- 04 — Hashing & anagram techniques
- 05 — String building perf (concat vs array-join, engine notes)
- 06 — Pattern matching intuition (KMP) with ASCII trace
- 07 — Trie for strings
- 08 — String parsing & stack-based decoding
- 09 — LeetCode practice set: String
